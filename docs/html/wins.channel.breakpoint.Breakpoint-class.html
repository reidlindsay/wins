Class Breakpoint

source code

Breakpoint pathloss model uses threshold policy for propagation model.

This channel model uses freespace pathloss up to a specified breakpoint distance. Beyond that distance (bpdist), the model uses a higher pathloss exponent (n):

PL (db) = PL0 + 10*n*log10(d/d0), (d>d0)

where PL0 is the pathloss at the breakpoint distance d0 (i.e. bpdist).
