Class ChannelModel

source code

Implement channel model between two ChannelInterface.

Packet Annotations and ChannelModel

ChannelModel may set any number of annotations to be used in determining a variety of parameters, including: packet error rate, path loss, signal to noise ratio, propagation delay, etc. The only annotation used by Channel is the 'cm-delay' annotation (see Packet Annotations and Channel).

Channel calls the apply() method before forwarding the packet from the source ChannelInterface to the destination.

filter(self, p, src, dst)

source code 

Filter out packets that the ChannelModel will drop.

By default this method does nothing (i.e. all packets pass through filter). Overload this in derived class to change filter.

Parameters:
  • p - Packet to filter.
  • src - Source ChannelInterface that sent p.
  • dst - Destination ChannelInterface.
Returns:
Reason for drop (string), or None if packet was not filtered out by the ChannelModel.

apply(self, p, src, dst)

source code 

Apply ChannelModel to specified packet.

By default this method returns p after setting the 'cm-delay' annotation to zero (i.e. denoting zero propagation delay).

Parameters:
  • p - Packet to be forwarded from src.
  • src - Source ChannelInterface that sent p.
  • dst - Destination ChannelInterface.
Returns:
Modified packet with appropriate annotations to be forwarded to destintation ChannelInterface by Channel.

Note: If this method returns None, then the packet will be dropped by the parent Channel in Channel.apply_model().
