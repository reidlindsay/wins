crcupdate(p, *args, **kwargs)

source code 

Add/update CRC32 for Packet p.

If p already has a CRC32, this method will overwrite the current CRC32; otherwise it will append a new one.

Parameters:
  • p - Packet to modify.
  • args - Arguments passed to CRC32 constructor if needed.
  • kwargs - Keywords passed to CRC32 constructor if needed.
Returns:
Modified Packet with updated CRC32.
