Class Port

source code

Interface for enabling connections between Element objects.

Port Targets

Each Port has a target. By default, each Port points to itself, thus a send() call on an unconnected Port will result in queueing of objects at the Port itself. Multiple Ports may be connected together in a chain. The target of any Port in this chain would be the Port at the tail of this chain. Ports can be connected using the member method Port.connect().
