rshnav(self, data, *args, **kwargs)

source code 

Calculate NAV value for RSH.

This method computes the RSH NAV as follows:

NAV = SIFS + DATA + SIFS + ACK
Parameters:
  • data - DATA packet.
  • args - Arguments passed on to DCF.rtsnav().
  • kwargs - Keywords passed on to DCF.rtsnav().
Returns:
Integer; representing NAV value.

Note: This method uses DCF.rtsnav() to calculate the new NAV value.
