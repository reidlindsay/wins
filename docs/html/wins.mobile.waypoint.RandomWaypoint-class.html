configure(self, pos=None, speed=None, pause=None, border=None, **kwargs)

source code 

Configure motion parameters.

The speed when traveling to the next waypoint is uniformly distributed in the specified range. Pause time at a waypoint is uniformly distributed in the specified range. The next waypoint is uniformly chosen at random from the area specified by the border.

If speed (or pause) are not tuples, they are assumed to be singleton values and represent constant speed (or pause) values.

Parameters:
  • speed - 2-tuple containing (min-speed, max-speed).
  • pause - 2-tuple containing (min-pause, max-pause).
  • border - 4-tuple containing (x-min, x-max, y-min, ymax).
