Class DSR

source code

Dynamic Source Routing protocol (RFC 4728).

This implementation uses link-level acknowledgements to do route maintenance (see MAC.drpdata and MAC.ackdata).

configure(self, mac=None, rreqrate=None, datarate=None, **kwargs)

source code 

Configure pointers and parameters.

If mac is not provided, this module will attempt to automatically determine the appropriate MAC.

If rreqrate and/or datarate are not specified, DSR will not attempt to set the rate annotation of the outgoing packet. It is important that the rreqrate be higher than datarate to ensure stable links for DSR.

Parameters:
  • mac - MAC module corresponding to this network protocol.
  • rreqrate - Rate index used for flooding RREQ messages.
  • datarate - Rate index used for sending other DSR messages.
  • kwargs - Additional keywords passed to Routing.configure().
Overrides: element.Element.configure

encap_data(self, p, src, dst, force=False, **kwargs)

source code 

Encapsulate data packet (if needed) for delivery to next hop.

If p is already an IP packet and force is false, this method will update 'src', 'dst', and 'ttl' fields as needed. Otherwise, this method encapsulates p with an IP header.

Parameters:
  • p - Packet to deliver.
  • src - Source address.
  • dst - Destination address.
  • force - If true, force encapsulation into a new IP packet.
  • kwargs - Additional keywords passed to IP constructor.
Returns:
Encapsulated/modified packet for delivery.
Overrides: protocol.routing.Routing.encap_data

Note: By default this method assumes that ptype indicates IPv4. Overload this method to handle other protocol types.
