Class RouteRequestTable

source code

DSR Route Request Table.

Manage route requests sent and forwarded by the network protocol. All FIFOs are implemented as lists where new objects are appended to the back and old objects are popped from the front.

Note: RouteRequestTable must be a child of a DSR module.
