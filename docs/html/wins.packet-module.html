Module packet

Contains classes for creating and modifying packets; contains Packet and ANNO.

Revision Info

  • $LastChangedBy: mandke $
  • $LastChangedDate: 2012-02-23 16:12:36 -0600 (Thu, 23 Feb 2012) $
  • $LastChangedRevision: 5420 $

Packet objects from Scapy now implicitly support an ANNO layer. They also have an id parameter (i.e. Packet._id), which is copied by copy() or deepcopy() operations.
