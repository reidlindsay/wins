Class CSMAC

source code

Base class for implementing carrier sense MAC protocols such as CSMA/CA.

This base class contains members to interact with carrier sense properties in a CSPHY. The actual implementation of this functionality is left to subclasses.

Note: In order to implement a CSMAC, a valid CSPHY must be set as its phy prior to starting. The set_phy() method has been overloaded to check that this condition has been satisfied.
