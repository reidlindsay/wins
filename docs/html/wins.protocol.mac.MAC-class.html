Class MAC

source code

Base class for medium-access control (i.e. Layer 2) protocols.

MAC Ports

Every MAC object has the following two ports:

  1. "TXU" - sends received packets to upstream protocol (usually ARP).
  2. "RXU" - receives traffic from an upstream element (usually ARP).

These ports are needed to properly interface MAC protocols with ARP. Additional ports should added by subclasses as needed.

MAC and Addressing

The format of MAC layer addresses depends on the hardware type enumeration htype. This parameter is used by ARP to determine how to properly format messages destined for a MAC layer or how to interpret messages delivered by a MAC layer. By default, MAC uses Ethernet addressing.

configure(self, phy=None, addr='auto', htype='auto', checkcrc=True, promiscuous=False, **kwargs)

source code 

Set up pointers, ports, and addressing.

By default, this method will use the automatic configuration option for set_address(). It will also set up the "TXU" and "RXU" ports.

Parameters:
  • phy - Pointer to PHY for MAC.
  • addr - Address for MAC (see MAC and Addressing for more).
  • htype - Hardware type enumeration for ARP; Ethernet by default.
  • checkcrc - Boolean flag; if false haserror() will ignore the CRC when determining if an error has occurred.
  • promiscuous - Boolean flag; should be used to indicate whether or not MAC is operating in promiscuous mode [default=False].

duration(self, p, *args, **kwargs)

source code 

Convenience method to get duration of packet p.

By default, this method calls PHY.duration() on the physical layer associated with this MAC (i.e. phy). If phy is not a valid PHY, this method will attempt to find and return the 'cif-duration' annotation.

Overload this method as needed.

Returns:
Duration of packet p in seconds.

Note: If duration cannot be determined, this method may raise an exception.

haserror(self, p, *args, **kwargs)

source code 

Convenience method to determine if packet p has an error.

By default, this method calls CRC32.haserror(). If no CRC32 layer is found, or no 'crcerror' annotation is found the packet is assumed to be error-free.

Overload this method as needed to change this operation.

Parameters:
  • args - Additional arguments passed to CRC32.haserror().
  • kwargs - Keyword arguments passed to CRC32.haserror().
Returns:
Boolean; true if error is found; false otherwise.

Note: This method ignores the CRC when checkcrc is set to false.

set_address(self, addr=None, htype=None)

source code 

Set address information.

If the parameter addr (or htype) is set to "auto", the address (or hardware type enumeration) will be determined automatically. If None is passed in, nothing is done.

Under automatic configuration, this method uses Ethernet frames and automatically configures the address using uid.

Parameters:
  • addr - Address for MAC.
  • htype - Hardware-type enumeration for ARP.
