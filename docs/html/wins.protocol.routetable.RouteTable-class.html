Class RouteTable

source code

Table to maintain routing information.

Each routing entry consists of a destination, route, and cost. This class supports multiple paths to the same destination. It uses cost to order the paths so that nexthop and route will select the lowest cost path.

addroute(self, dest, nexthop=None, cost=None)

source code 

Add new route to dest.

If nexthop is None, it is assumed that dest is a one hop neighbor. If cost is None, RouteTable() uses cost() to determine the cost of the route through nexthop.

Parameters:

  • nexthop - Address of nexthop [default=`None`].
  • cost - Cost of route [default=None].

getcost(self, dest, nexthop, ts)

source code 

Determine cost of path to dest using route through nexthop.

By default this method calculates the cost using the timestamp ts. That is, this method returns the "age" of the path.

Parameters:
  • dest - Destination of path.
  • nexthop - Next hop in path.
  • ts - Timestamp for when route was added to table.

Note: Overload this method to change how cost of a path is computed.
